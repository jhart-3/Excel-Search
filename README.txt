# Excel Search App

A Python application to search and retrieve data from Excel files with a GUI. Users can configure search parameters, view results in a table, and export them.

## Features

- Configure search parameters
- User-friendly GUI
- View search results in a table
- Export results to Excel

## Prerequisites

- Python 3.x
- PyQt5
- pandas
- openpyxl

## Usage

1. Run the application (`excel_search_app.py`).
2. Choose an Excel file.
3. Configure search parameters.
4. Enter a search phrase and click "Search".
5. View and export search results.
